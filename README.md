# Дипломный проект курса «Fullstack-разработчик на JavaScript FJS-71» фронт

## Фронт сайта-агрегатора просмотра и бронирования гостиниц

### Исполнитель Серебренников Кирилл Викторович

---

### Описание структуры папок и файлов проекта

Всего в фронте девять папок.
Кроме этого в корне проекта размещен файл `server.js` - сервер для подгрузки скомпилированного кода.

### Папка `components`

В папке собраны отдельные компоненты используемые в процессе создания сайта.
В ней в свою очередь, находятся индексный файл для экспорта и 11 папок отдельных компонентов:

- `Custom` - папка в которой содержаться файлы кастомных элементов:
  - `CustomButton.tsx` - файл кастомной кнопки;
  - `ErrorLoad.tsx` - файл отображения иконки загрузки или ошибки;
  - `Icons.tsx` - файл с используемыми иконками;
  - `InputField.tsx` - файл кастомного поля ввода;
- `Header` - папка в которой содержится заготовок сайта;
  - Header.tsx - файл верхней части сайта;
- `Hotel` - папка с компонентами страницы Отели и Комнаты:
  - `FindHotelForm.tsx` - форма поиска отеля;
  - `HotelForm.tsx` - форма создания и редактирования отеля;
  - `HotelListForm.tsx` - элемент списка отелей;
  - `RoomDescription.tsx` - элемент просмотра комнаты;
  - `RoomForm.tsx` - форма создания и редактирования комнаты;
- `Layout` - папка макета страницы с навигацией
  - `Layout.tsx` - фаблон страницы с навигацией
- `Login` - папка с формами входа и регистрации пользователей
  - `LoginForm.tsx` - форма входа;
  - `RegisterForm.tsx` - форма регистрации нового пользователя;
- `LoginNav` - папка элемента выбора на навигационном меню
  - `LoginSelect.tsx` - элемент выбора между входом или регистрацией на навигационном меню
- `Logo` - папка с логотипом
  - `Logo.tsx` - файл логотипа
- `Nav` - папка с элементами навигации;
  - `Nav.tsx` - файл навигации;
  - `Pagination.tsx` - файл элемента пагинации;
- `Reservation` - папка с элементами регистрации
  - `ClientReservationForm.tsx` - файл формы бронирования гостиницы
- `SupportRequest` - папка с эленментами обращений
  - `MessageForm.tsx` - форма отправки сообщений
- `Users` - папка с элементами пользователей
  - `FindUserForm.tsx` - файл формы поиска пользователей
  - `RoleEnumSelect.tsx` - файл поля со списком выбора роли пользователя
  - `UserListForm.tsx` - файл формы списка пользователей

### Папка `constants`

В папке содержаться константы, которые используются в приложении.
Папка состоит из индексного файла для экспорта и трех папок:

- `Errors` - папка ошибок
  - `Errors.ts` - файл содержит константы возможных ошибок;
- `login` - папка с константами входа в приложение
  - `loginLinks.ts` - файл ссылок при входе;
  - `logoutLinks.ts` - файл ссылок при выходе из приложения;
  - `roles.ts` - файл с константами ролей пользователя;
- `View` - папка с константами отображения информации;
  - `ViewConstants.ts` - файл с константами отображения информации

### Папка `context`

В папке содержится контекст для использования данных во всем приложении

- `AuthContext.ts` - в файле содержится контекст

### Папка `hooks`

В папке содержатся кастомные хуки:

- `useAuth.ts` - хук авторизации;
- `useChat.ts` - хук чата;
- `useGetData.ts` - хук получения информации через API

### Папка `interfaces`

В ней содержатся интерфейсы, используемые в приложении.

### Папка `pages`

В папке содержатся страницы приложения

- `Hotels` - в папке содержаться страницы, связанные с гостиницами и комнатами

  - `FindHotel.tsx` - файл страницы поика гостиницы;
  - `Hotel.tsx` - файл страницы гостиницы;
  - `HotelsList.tsx` - файл страницы списка гостиниц;
  - `Room.tsx` - файл страницы комнаты в гостинице;
  - `RoomEdit.tsx` - файл страницы редактирования гостиницы;

- `Login` - в папке содержаться страницы регистрации и идентификации пользователя
  - `Login.tsx` - файл со страницей входа в приложение;
  - `Register.tsx` - файл со страницей регистрации нового пользователя
- `Reservation` - папка со страницамибронирования комнат
  - `ClientReservationList.tsx` - файл со страницей зарезервированных клиентом комнат
- `SupportRequests` - папка в которой содержаться страницы обращений клиентов в поддержку
  - `Chat.tsx` - страница чата;
  - `NewChat.tsx` - страница создания нового чата;
  - `SupportRequests.tsx` - страница списков чатов;
- `Users` - папка со страницами, относящихся к работе с пользователями
  - `AddNewUser.tsx` - страница добавления нового пользователя
  - `UsersList.tsx` - страница списка пользователей

### Папка `providers`

В папке содержатся файлы провайдера авторизации

### Папка `redux`

- `slices` - папка содержит слайсы redux
  - `apiSlice.ts` - слайс сохранения информации при доступе к внешним API;
  - `authSlice.ts` - слайс данных авторизации;
  - `dateSlice.ts` - слайс бронирования на определенную дату;
  - `navSlice.ts` - слайс навигации;
  - `tempSlice.ts`- слайс временных данных;
- `store` - папка с описание хранидлища данных
- `thunks` - папка thunk функций

  - `AuthThunk.ts` - функция авторизации;
  - `GetDataFromApiThunk.ts` - функция получение данных через API;
  - `LogoutThunk.ts`- функция выхода из приложения;
  - `NavThunk.ts` - функция запроса навигации;

  ### Папка `scripts`

  В папке собраны функции используемые в приложении

- `Auth` - папка функций автроизации
  - `SetAuth.ts` - функция авторизации;
  - `SetLogout.ts` - функция выхода из приложения;
  - `SetRegister.ts` - функция регистрации;
- `Custom` - папка кастомных функций
  - `ConvertDate.ts` - функция преобразования даты;
- `Errors`- папка с функциями обработки ошибок
  - `GetError.ts` - функция преобразования кодов ошибок
- `GetData` - папка с функцииями получения данных
  - `GetData.ts` - функция получения данных;
  - `GetDataFromAPI.ts` - функция получения данных;
  - `GetFileFromLink.ts` - получение файла по ссылке;
  - `GetFileNameFromLink.ts` - получение имени файла по ссылке;
  - `SendFormDataToAPI.ts`- отправка данных;
- `Hotel` - папка с функциями гостиницы и комнаты
  - `CreateHotel.ts` - создание гостиницы;
  - `CreateRoom.ts` - создание комнаты;
  - `GetAllHotels.ts` -получение админеистратором списка всех гостиниц;
  - `GetHotelById.ts` - получение гостиницы по ID;
  - `GetHotels.ts` - получение пользователями списка отелей;
  - `GetRoomsByHotelId.ts` - получение комнат по ID отеля;
  - `UpdateHotel.ts` - редактирование отеля;
  - `UpdateRoom.ts` - редактирование комнаты;
- `Nav` - впапке находятся скрипты связанные с навигацией
  - `GetNav.ts` - получение доступных для пользователя ссылок;
- `Reservation` - папка со скриптами бронирования отеля
  - `DeleteReservationById.ts` - удаление бронирования по ID бронирования;
  - `GetReservation.ts` - получение пользователем списка зарезервированных комнат;
  - `GetUserReservationById.ts` - получение менеджером списка зарезервированного пользователем комнат;
  - `SetReservation.ts` - бронирование комнаты;
- `Router` папка со скриптом навигации по страницам приложения
  - `GetRoute.tsx` - переход на страницу по навигации;
- `SupportRequests` - папка со скриптами обращений
  - `CreateChat.ts` - создание обращения;
  - `GetChatMessages.ts` - получение списка сообщений чата;
  - `GetClientSupportRequests.ts` - получение клиентом списка своих обращений;
  - `GetManagerSupportRequests.ts` - получение менеджером обращений клиентов;
  - `MarkMessagesAsRead.ts` - отметка сообщений прочитанными;
  - `SendMessage.ts` отправка сообщения;
- `User` - папка со скриптами пользователей
  - `AddUser.ts` - добавление нового пользователя администратором;
  - `GetUsers.ts` - получение администратором или менеджером списка пользователей;
